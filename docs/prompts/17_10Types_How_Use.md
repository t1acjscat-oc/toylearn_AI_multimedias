## 프롬프트 유형과 선택 이유

| 프롬프트 유형                                 | 선택 이유                                                                                 |
| --------------------------------------- | ------------------------------------------------------------------------------------- |
| **역할 지시형(Role Prompt)**                 | 사용자가 '멀티미디어 콘텐츠 매니저' 역할을 수행하며 전문적인 관점에서 문제를 해결해야 하기 때문                                |
| **상황 기반(Situation-Based Prompt)**       | 실제 마케팅 환경과 유사한 맥락(예산 제한, 클라이언트 요구, 플랫폼별 특성 등)을 주고 현실적인 결과 도출 필요                       |
| **결과물 지향형(Task-Oriented Prompt)**       | 단순 제안이 아니라 **각 플랫폼에 맞는 구체적 콘텐츠 변환 전략 및 실행 항목**이 결과물로 요구됨                              |
| **멀티 출처 활용형(Multi-modal Input Prompt)** | 하나의 영상 소스를 다양한 플랫폼에 맞게 변형(Re-purpose)해야 하므로, **“주어진 자원을 기반으로 새로운 콘텐츠를 재구성”**하는 유형에 적합 |
| **체계적 출력 요구형(Format-Specified Prompt)** | 플랫폼별로 정리된 결과가 필요하므로 표/항목 기반으로 출력되도록 명확히 요구                                            |

## 작성 위해 추가 필요 요건

| 필요 요소                                              | 포함 이유                                                                             |
| -------------------------------------------------- | --------------------------------------------------------------------------------- |
| 📌 **원본 콘텐츠 설명 방식** (예: 요약, 주요 장면, 나레이션 스크립트 분석 등) | 영상 내용 기반으로 플랫폼별 재구성이 가능하도록 명확한 정보 필요                                              |
| 📌 **각 플랫폼 콘텐츠 형식 기준**                             | YouTube(롱폼), Instagram Reels(15~60초), TikTok(세로 9:16 단문), 블로그(텍스트+이미지 중심) 등 규격 반영 |
| 📌 **콘텐츠 변환 규칙 명시** (길이, 톤, CTA, 해시태그)             | AI가 단순 요약이 아니라 플랫폼에 최적화된 콘텐츠를 “재가공”할 수 있게 함                                       |


## 가상 문제 상황
1. 가상 문제 상황

페르소나: 미디어 에이전시의 '멀티미디어 콘텐츠 매니저'

배경: 당신은 최근 대형 식품 기업의 신규 '친환경 밀키트' 브랜드 런칭 캠페인을 담당하게 되었습니다. 클라이언트(식품 기업)는 제품의 개발 과정, 재료의 신선함, 조리 편의성을 강조하는 **15분 분량의 고화질 홍보 영상(원본 소스)**을 대량으로 제작하여 전달했습니다.

핵심 문제: 클라이언트는 이 15분짜리 원본 영상 소스를 활용하여 유튜브(롱폼), 인스타그램(릴스/캐러셀), 틱톡, 네이버 블로그 등 모든 소셜 채널에 동일하게 배포하기를 원합니다. 하지만 당신은 플랫폼별 특성과 사용자 소비 행태가 완전히 다르다는 것을 알고 있습니다. 15분짜리 영상을 릴스나 틱톡에 그대로 올릴 수는 없으며, 블로그에는 영상과 다른 텍스트 중심의 콘텐츠가 필요합니다.

해결 과제: 클라이언트의 예산은 한정되어 있어 추가 촬영은 불가능합니다. 당신은 오직 제공된 15분짜리 원본 영상(과 그 스크립트)만을 활용하여, 각기 다른 4개 플랫폼(유튜브, 인스타그램 릴스, 
틱톡, 네이버 블로그)의 성격에 최적화된 '멀티미디어 콘텐츠 배포 전략'을 수립하고, 각 플랫폼에 맞는 콘텐츠 변형(Repurposing) 기획안을 구체적으로 생성해야 합니다.

2. 훈련 초점 (힌트)

이 문제는 [콘텐츠 변형 및 재가공] 능력을 평가하기에 적합합니다.
