## 프롬프트 유형과 선택 이유

유형: 문제 해결형 프롬프트 (Problem-Solving Prompt) + 원인 분석형(Why형)

| 선택 이유                                         | 설명                                             |
| --------------------------------------------- | ---------------------------------------------- |
| 문제 상황이 “성과 저조의 원인을 명확히 분석하고 대안을 제시해야 하는 상황”   | 단순 요약이나 아이디어 생성이 아니라 **구체적 원인 분석과 해결 전략**이 요구됨 |
| 링크에서 제시된 “왜(Why)형 프롬프트 / 원인 분석형 프롬프트”가 정확히 일치 | “왜 성과가 낮았는지 분석하라, 개선안을 제시하라”는 요청에 가장 부합        |
| 출력물이 실제 회의 보고서로 활용될 가능성이 높음                   | 따라서 **논리적 구조 + 근거 기반 분석 + 실행 가능한 해결책**을 도출해야 함 |

## 작성 위해 추가 필요 요건

| 항목                                          | 필요한 이유                             |
| ------------------------------------------- | ---------------------------------- |
| C 광고 영상의 스크립트 또는 주요 메시지                     | 왜 이탈률이 높은지, 표현 방식/카피 문구 문제인지 파악 필요 |
| 타겟층 정보(연령, 성별, 뷰티 관심도)                      | 할인 혜택이 타겟에게 매력적이지 않은 이유 확인 가능      |
| 광고 집행 채널/노출 위치 (YouTube TrueView, Shorts 등) | 플랫폼별 시청 패턴이 다르기 때문에 필수             |
| 경쟁사 동일 시기 광고 유무                             | 상대적 비교 분석 가능                       |
| CTA(Call To Action) 문구 및 노출 타이밍             | 구매 유도 타이밍이 너무 늦거나 약할 수 있음          |
| 썸네일/첫 3초 구성 정보                              | 이탈률 80%의 핵심 원인 탐색                  |

## 가상 문제 상황

1. 가상 문제 상황
페르소나: 광고 대행사의 콘텐츠 제작 팀장
배경: 당신은 클라이언트 'A'(신규 뷰티 브랜드)의 신제품 런칭을 위해 3가지 버전의 유튜브 범퍼 광고(6초) 영상을 제작하여 집행했습니다. 3가지 영상은 각각 다른 소구점(A: 성분 강조, B: 사용 후기, C: 할인 혜택)을 강조했습니다.
핵심 문제: 캠페인 2주차 데이터 분석 결과, A와 B 영상은 평균 클릭률(CTR) 2.5%, **구매 전환율 1.8%**를 기록하며 순항 중입니다. 하지만 **가장 많은 예산을 투입한 C 영상(할인 혜택 강조)의 CTR은 0.8%**에 불과하며, **구매 전환율은 0.1%**로 처참한 수준입니다. 특히 C 영상은 3초 이내 이탈률이 80%에 육박합니다.
해결 과제: 클라이언트는 C 영상의 즉각적인 중단과 예산 환불까지 고려하고 있습니다. 당신은 C 영상의 성과가 저조한 구체적인 원인을 분석하고, 이 영상을 즉시 수정(re-edit)하거나 대체할 수 있는 긴급 대응 방안을 내일 오전 회의까지 보고해야 합니다.

2. 훈련 초점 (힌트)
이 문제는 [원인 분석] 능력을 평가

# ✅ C 영상의 성과 저조 원인 분석 및 개선 전략

## **1. 문제 요약**

* C 영상은 **할인 혜택을 강조했음에도 CTR 0.8%, 구매전환율 0.1%, 3초 이탈률 80%**로 매우 저조함.
* A/B 영상 대비 클릭과 전환율이 1/3 이하로 떨어졌고, **광고 첫 3초 안에서 대부분 이탈**.
* **핵심 실패 요인:** “혜택 전달”보다 “관심 유발”에 실패했으며, 브랜드·제품 매력보다 “할인”을 전면에 내세운 전략적 오류.

---

## **2. 원인 분석 (카테고리별 / A·B 비교 포함)**

| 카테고리                       | C 버전 문제점                                              | A·B 대비 차이                   |
| -------------------------- | ----------------------------------------------------- | --------------------------- |
| **메시지/카피 문제**              | "지금 할인!" 같은 무감각한 혜택 중심 카피 → 차별성·신뢰성 부족                | A/B는 성분·후기를 통해 니즈→신뢰→행동 유도  |
| **시각적 / 첫 3초 구성**          | 첫 장면부터 숫자/텍스트 나열 → 정서·흥미 자극 0                         | A/B는 피부 클로즈업·사용 장면 등 감각적 영상 |
| **타겟-콘텐츠 부적합**             | 할인을 중심으로 하지만 “새로운 브랜드”라 신뢰 기반 부족 → 소비자가 할인보다 신뢰 먼저 필요 | A/B는 브랜드 신뢰를 구성 후 정보 제공     |
| **CTA(Call To Action) 문제** | “지금 바로 구매” 같은 직설 CTA → 구매장벽 낮추기 실패                    | A/B는 구매 전 단계인 ‘신뢰·공감’에 초점   |
| **플랫폼/포맷 적합성**             | 6초 범퍼 특성상 설득형 정보 전달 어려운데, 설명형 구조                      | A/B는 감각+직관형 구성이어서 범퍼에 적합    |

---

## **3. 즉시 수정 가능한 Re-edit 방안 (3가지)**

| 방안                          | 목표          | 핵심 카피                | **첫 3초 구성**                                                     | 기대 효과                  |
| --------------------------- | ----------- | -------------------- | --------------------------------------------------------------- | ---------------------- |
| **1. 감각적 할인+사용 장면 결합**      | 이탈률↓, 관심↑   | “바르자마자, 매끈. 지금 할인 중” | 0~1초: 피부 클로즈업 → 1~3초: 바르는 순간 광채 → 3~6초: “오늘만 -20%”              | 감각+혜택 결합으로 주목도 상승      |
| **2. 혜택보다 ‘왜 싸게 주는가’ 스토리화** | 신뢰 확보       | “신규런칭 기념, 첫 고객에게만”   | 모델 등장 + 짧은 시선 처리 → 로고 → “첫 1만명만 할인”                             | ‘이유 있는 할인’으로 진정성 강화    |
| **3. 기존 A/B와 C 요소 하이브리드**   | B 수준 CTR 회복 | “후기 인정템, 지금은 할인 중”   | 0~2초: 만족 후기 자막 삽입 → 2~4초: 피부 개선 before/after → 4~6초: “공식몰 -20%” | 후기 신뢰 + 혜택 결합으로 전환율 상승 |

---

## **4. 완전히 새로운 대체 콘셉트 아이디어 (2+)**

### ✅ 콘셉트 1: “할인 안 하면 손해” FOMO 심리 자극

* **주요 카피:**
  “품절 전에 사둘걸… 지금도 늦진 않았어”
* **연출 방식 (6초)**

  * 0~2초: “작년엔 품절돼서 못 샀어요” 후기 스타일 컷
  * 2~4초: 제품 사용 장면 + 자연광
  * 4~6초: “이번엔 할인 중. 놓치면 또 후회”
* **예상 효과:** 긴급성+사회적 증거 → 클릭 유도↑, FOMO 기반 전환 유도

### ✅ 콘셉트 2: 비교 체감형 (Before vs After + 혜택)

* **주요 카피:**
  “Before▶After, 지금은 할인까지”
* **연출 방식 (6초)**

  * 0~2초: 피부 트러블/건조 before
  * 2~4초: after + 광채 피부 클로즈업
  * 4~6초: “정가 29,000 → 오늘 23,000”
* **예상 효과:** 시각적 변화 → 감각적 설득 → 가격 정보로 확실한 행동 유도

---

## **5. 클라이언트 보고용 결론 및 리스크 코멘트**

**결론 핵심 메시지**

* 현재 저조한 성과는 **“할인 자체의 문제”가 아니라 “브랜드 신뢰와 감각적 흡입력 없이 단독으로 할인을 전면 배치한 전략적 문제”**에서 비롯됨.
* A/B와 동일한 구조의 감각/신뢰 요소를 C에도 결합하면 **CTR 최소 2% 회복 가능**.
* 즉시 편집 가능한 Re-edit으로 **예산 추가 없이 개선 가능**.

**리스크 및 예산 환불 요청 대응 논리**

* 매체/타겟 설정은 동일 조건에서 A/B는 정상 성과 → **제작 메시지 문제이지 매체 낭비가 아님**.
* 원본영상 활용 Re-edit 가능 → **추가 제작비 없이 퍼포먼스 개선 가능**, 즉 환불보다 수정이 효율적.
