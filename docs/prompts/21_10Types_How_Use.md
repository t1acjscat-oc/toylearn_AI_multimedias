## 프롬프트 유형과 선택 이유
텍스트 요약 및 매체 변환형 프롬프트(Text-to-Script Transformation Prompt)

## 작성 위해 추가 필요 요건

| 구분           | 구체적 요구사항                                                                  |
| ------------ | ------------------------------------------------------------------------- |
| **입력 데이터**   | 10페이지 분량의 ‘생성형 AI의 미래와 윤리적 쟁점’ 리포트 (전문적이고 학술적인 어투)                        |
| **출력 형식**    | 60초 내외의 숏폼 영상 촬영 대본 (Hooking 멘트, 내레이션, 자막, 시각효과 지시 포함)                    |
| **타깃 수준**    | 초등학생도 이해 가능한 쉬운 언어                                                        |
| **영상 톤앤매너**  | 빠르고 명쾌하며 호기심을 유발하는 톤 (틱톡/릴스 스타일)                                          |
| **핵심 조건**    | - 초반 3초 Hook 필수<br>- 중간부 정보 요약(핵심 개념 3가지)<br>- 엔딩 콜투액션 포함("더 알고 싶다면 구독!") |
| **출력 세부 구성** | `scene`, `visual`, `subtitle`, `narration`, `sound` 5개 항목으로 구체화           |

## 가상 문제 상황
1. 가상 문제 상황

👤 페르소나: 뉴미디어 스타트업의 '숏폼 콘텐츠 PD'

🏙️ 배경: 당신은 IT 및 테크 트렌드를 소개하는 유튜브 채널을 운영하는 스타트업 '테크인사이드'의 PD입니다. 최근 회사는 틱톡과 릴스 유입을 늘리기 위해 '60초 지식 요약' 시리즈를 런칭했습니다.

🚨 핵심 문제: 마케팅 팀으로부터 **"생성형 AI의 미래와 윤리적 쟁점"**이라는 주제의 10페이지짜리 심층 분석 리포트(PDF)를 전달받았습니다. 내용은 매우 전문적이고 딱딱한 텍스트 위주여서, 이를 그대로 읽으면 시청 지속 시간이 3초 미만으로 떨어질 것이 확실시되는 상황입니다.

🎯 해결 과제:
복잡하고 긴 텍스트 리포트 내용을 대중이 이해하기 쉬운 '초등학생도 이해하는 수준'의 언어로 변환해야 합니다.
단순 요약을 넘어, 시각적 요소(화면 구성, 자막, 효과음)가 포함된 **60초 분량의 숏폼 영상 촬영 대본(Script)**을 완성해야 합니다. (특히, 초반 3초 안에 시선을 끄는 '후킹(Hooking)' 멘트가 필수입니다.)
출력은 JSON 형식

2. 훈련 초점 (힌트)
이 문제는 [복잡한 텍스트의 요약 및 매체 변환(Text-to-Script)] 능력을 평가
단순히 내용을 줄이는 것이 아니라, **영상 문법(Visual Cues)**에 맞게 내용을 재구성하는 프롬프트 엔지니어링이 필요.

## 문제 해결 프롬프트 

```
{
  "prompt_title": "AI 윤리 리포트를 60초 숏폼 영상 스크립트로 변환",
  "persona": "당신은 뉴미디어 스타트업 '테크인사이드'의 숏폼 콘텐츠 PD입니다.",
  "goal": "복잡한 10페이지 리포트를 초등학생도 이해할 수 있는 언어로 재구성하여, 시청 지속 시간이 높은 60초 분량의 숏폼 영상 대본을 만드세요.",
  "input_format": {
    "source": "‘생성형 AI의 미래와 윤리적 쟁점’ 리포트의 전문 텍스트",
    "length": "약 10페이지"
  },
  "output_format": {
    "type": "숏폼 영상 촬영 대본(JSON 구조)",
    "structure": [
      "scene: 장면 번호",
      "visual: 화면 구성 설명(배경, 인물, 그래픽 등)",
      "subtitle: 자막 텍스트",
      "narration: 내레이션 대사",
      "sound: 효과음 또는 BGM 제안"
    ]
  },
  "instructions": {
    "1": "리포트의 핵심 개념 3개를 중심으로 내용을 단순화하세요.",
    "2": "전문 용어는 반드시 비유나 예시로 바꾸세요.",
    "3": "영상의 첫 3초에 강력한 Hooking 멘트를 삽입하세요.",
    "4": "중간부는 15초 단위로 리듬감 있게 구성하세요.",
    "5": "마지막 5초는 시청자 참여 유도형 엔딩으로 마무리하세요.",
    "6": "전체 대본은 60초 분량으로 제한하고, 한 장면당 10초 이내로 구성하세요."
  },
  "output_example": {
    "scene": 1,
    "visual": "AI 로봇이 눈을 반짝이며 등장",
    "subtitle": "AI가 인간보다 똑똑해진다면?",
    "narration": "AI가 그림도 그리고, 음악도 만들어요. 그런데… 우리가 통제할 수 있을까요?",
    "sound": "짧은 전자음 + 긴장감 있는 BGM"
  },
  "tone": "명확하고 흥미로운 설명형, 초등학생도 이해 가능한 쉬운 언어",
  "length_limit": "60초"
}
```

 **핵심 구조**

* **Hook (0~3초)**: “AI가 시를 쓰고, 그림을 그리고, 노래까지 만든다고?!”
* **Main Point 1 (3~18초)**: 생성형 AI란 무엇인가 (쉽고 재밌게)
* **Main Point 2 (18~33초)**: 가짜 영상·정보 문제 (윤리적 쟁점 1)
* **Main Point 3 (33~50초)**: 진짜/가짜 구분과 AI 윤리의 필요성 (윤리적 쟁점 2, 3)
* **CTA (50~60초)**: 구독 유도 및 다음 콘텐츠 예고
